# sudoku-solver
sudoku-solver

## How run

Execute that commands in your terminal:

```shell
    git clone https://github.com/altherlex/sudoku-solver.git
    cd sudoku-solver
    bundle install
    irb
```
```ruby
    require './sudoku-solver'
    game = Sudoku.new
    // You see the building solution frame by frame
    game.next
```

## Requirements

    brew install ruby
    brew install git
    gem install bundle
